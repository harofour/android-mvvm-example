## 시작하기
1. `git clone https://github.com/harofour/android-mvvm-example.git`  
2. API 키를 발급하여 *res/values/string.xml* 에 추가  
```html
<resources>  
	<string name="api_key">Enter Your APIKEY</string>  
</resources>
```
